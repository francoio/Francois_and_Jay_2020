
## README

In an admixtools analysis, three files with the following names are needed in this folder:
* snps.geno
* snps.snp
* snps.ind

To create these files, copy the following files 
* geno_filter.geno
* snp_filter.snp
* ind_filter.ind

from the "Ancient_DNA" zipped folder, and rename them with the above names. Then remove this file from the folder.
